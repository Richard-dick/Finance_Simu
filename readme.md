### 手续费

买入手续费为
- 佣金的0.01%（万1）
- 过户费的0.001%（万0.1）
- 规费的0.005%（万0.5）
卖出手续费为
- 佣金的0.01%（万1）
- 过户费的0.001%（万0.1）
- 印花税的0.05%（万5）

综合来看，我们将两种手续费和一下，大概是万7的水平，不做免5

```python
def sell_fee(amount):
    """
    计算卖出手续费，抽象后只是用这个
    :param amount: 卖出金额
    :return: 手续费
    """
    return max(amount * 0.0007, 5)
```
